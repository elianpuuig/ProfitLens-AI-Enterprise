# 💎 ProfitLens AI - Business Intelligence & IA Strategy

ProfitLens AI transforma datos crudos en decisiones de alta rentabilidad. Diseñado para empresas que buscan escalar mediante auditoría de datos, análisis de Pareto avanzado y proyecciones de Forecast con Inteligencia Artificial.

## ✨ Características Principales
- **Auditoría IA:** Limpieza y normalización automática de bases de datos.
- **Data Engine:** Conexión nativa con Excel, CSV y SQL.
- **Consultoría Gold:** Reportes estratégicos generados por LLMs basados en tus métricas reales.
- **Forecast IA:** Predicción de ventas para el próximo mes con modelos de regresión.

## 🛠️ Tech Stack
- **Frontend:** Streamlit (Custom Cyber-Dark UI)
- **Engine:** Python, Pandas, Plotly, XlsxWriter
- **Auth & DB:** Firebase & SQLite
- **Payments:** Stripe Integration
